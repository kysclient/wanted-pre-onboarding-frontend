# wanted-pre-onboarding-frontend

이 레파지토리는 원티드 프리온보딩 프론트엔드 과정 선발 과제를 위해 만들어졌습니다.


## Available Scripts

In the project directory, you can run:

### `yarn install`

then

### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
